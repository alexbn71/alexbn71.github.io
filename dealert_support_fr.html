❓ Questions fréquentes

Comment scanner un coupon ?
Allez dans l'onglet Scanner, pointez l'appareil photo sur le coupon et appuyez sur le bouton de prise de vue. L'IA détecte automatiquement le magasin, le montant et la date d'expiration. Vous pouvez également importer une image depuis votre galerie.

Je ne reçois pas les notifications de proximité. Que faire ?
Assurez-vous d'avoir accordé l'autorisation de localisation « Toujours » dans les paramètres système (Réglages → Dealert → Localisation → Toujours). Vérifiez également que les notifications sont activées.

Puis-je utiliser Dealert sans appareil photo ?
Oui. Vous pouvez importer des images depuis votre galerie ou saisir les données manuellement via le bouton « Saisie manuelle » dans l'onglet Scanner.

Mes données sont-elles en sécurité ?
Toutes les données (coupons, images, localisations) sont enregistrées exclusivement sur votre appareil. Nous n'utilisons pas de serveurs externes. Pour plus de détails, consultez notre Politique de confidentialité.

Comment supprimer toutes mes données ?
Désinstallez l'application : toutes les données sont automatiquement supprimées de votre appareil.

Qu'est-ce que Dealert Premium ?
Le plan gratuit vous permet de sauvegarder un nombre limité de coupons actifs. Premium supprime cette limite. Vous pouvez vous abonner depuis l'onglet Paramètres.

Comment annuler mon abonnement Premium ?
Sur iPhone : Réglages → Votre nom → Abonnements → Dealert → Annuler l'abonnement.
